# Esra Yılmaz - Software Engineer Portfolio 👩‍💻

Bu proje; **.NET Core** ve **React** teknolojilerinde uzmanlaşan bir Yazılım Mühendisi olan **Esra Yılmaz**'ın kişisel portfolyo web sitesidir.
Modern, karanlık mod (dark theme) tasarımıyla; teknik yetkinlikleri, projeleri, çalışma metodolojisini ve mühendislik vizyonunu profesyonel bir şekilde sunar.

## 🌐 Proje Özeti

Site, klasik bir CV'nin ötesine geçerek; ziyaretçilere (Recruiter ve Teknik Liderler) adayın mühendislik disiplinini, kod kalitesine verdiği önemi ve kullandığı teknoloji yığınını etkileşimli bir şekilde aktarmak için tasarlanmıştır.

## ✨ Öne Çıkan Özellikler

- 🏠 **Hero Bölümü:** Yazılım Mühendisi unvanı, vizyonu ve sosyal medya/GitHub bağlantıları.
- 🎓 **Eğitim & Deneyim:** Sakarya Üniversitesi, İSTKA Techcareer Bootcamp ve Enoca Akademi süreçlerinin ikonlarla görselleştirilmiş sunumu.
- 🛠 **Teknoloji Yığını (Tech Stack):** - **Backend:** .NET Core, C#, Entity Framework
  - **Frontend:** React, Next.js, HTML5, CSS3, Bootstrap 5
- 📂 **Projeler:** GitHub entegrasyonlu, seçilmiş Full-Stack ve Frontend projeleri.
- 🧠 **Çalışma Metodolojisi (Sıkça Sorulanlar):** Mühendislik yaklaşımı, Agile/Scrum uyumu, Clean Code prensipleri ve Güvenlik/Performans standartları.
- 🌓 **Modern Dark UI:** Yazılım dünyasına uygun, göz yormayan, kontrastı yüksek özel renk paleti.
- 🎞️ **Animasyonlar:** AOS (Animate On Scroll) kütüphanesi ile akıcı sayfa geçişleri.
- 📱 **Tam Responsive:** Mobil, tablet ve masaüstü uyumlu Bootstrap 5 ızgara sistemi.

## 🛠 Kullanılan Teknolojiler

Proje, modern web standartlarına uygun olarak aşağıdaki teknolojilerle geliştirilmiştir:

- **HTML5** (Semantik Yapı)
- **CSS3** (Özel Değişkenler & Flexbox/Grid)
- **Bootstrap 5** (Responsive İskelet)
- **Vanilla JavaScript** (DOM Manipülasyonu)
- **AOS** (Scroll Animasyonları)
- **Font Awesome 6** (İkon Seti)
- **Devicon** (Teknoloji Logoları)
- **Google Fonts** (Poppins & Inter)
